## プロジェクト概要
Vanilla HTML / CSS で構築するシンプルなランディングページ（LP）。

## 技術スタック
- HTML5
- CSS3 (css/style.css に配置する。インラインスタイルは非推奨)

## 開発・Git運用ルール（※超重要）
- 環境: WSL (Linux) 上で開発している。
- デプロイ: GitHub Pages (mainブランチ) を使用。
- Git操作: セキュリティ（SSH署名）の都合上、Claude はコードの編集のみを行い、`git commit` および `git push` は絶対に自動実行しないこと。必ずユーザーに手動で実行させるよう促すこと。
- 言語: 追加するコンテンツのテキストやコード内のコメントは、すべて日本語を使用すること。
